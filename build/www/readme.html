Another BonziWORLD

Leading the industry in literally any chat client.

What the hell is this?

Another BonziWORLD: ItzCrazyChace Edition is a node.js and socket.io client featuring a purple ape named Bonzi. The client also uses speak.js as the text-to-speech voice. It delays for a bit, but hey, it works.

Why did you make this? Why WOULD you make this?

Memes. Also because I'd already finished the BonziBUDDY portion of this project like a year in advance and was trying to find some way to put it to use.

Why did you make a server?

This server was most likely inspired by Glitch56 or some random user who had Another BonziWORLD. I had to make this for some random reason.

How do I use it?

  1. Enter a nickname. If you don't enter one, you'll be most likely named "Anonymous". (we r legion du nut 4 get)
  2. Optionally, enter a room ID. If you want to join someone else, their room ID will be in the bottom-right corner of the screen.
    • If you enter a room ID that doesn't exist, you will be placed in a private room which can only be joined by people you share the ID with.
  3. Be a BonziBUDDY.
